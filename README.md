# 🔐 SOC Threat Intelligence Analyzer

A SOC-focused threat intelligence dashboard built using Python and Streamlit that analyzes IP reputation using VirusTotal and AbuseIPDB APIs.

---

## 🚀 Features

- VirusTotal API integration
- AbuseIPDB API integration
- Threat level classification
- IP reputation analysis
- Country and organization detection
- Downloadable threat reports
- Scan history tracking
- Interactive Streamlit dashboard

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Requests API

---

## ▶️ How to Run

1. Clone the repository

2. install
3. pip install -r requirements.txt
4. Add your API keys inside app.py
VirusTotal API Key
AbuseIPDB API Key
Run the project
